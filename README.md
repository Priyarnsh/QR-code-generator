# Modern QR Code Generator

A fully featured, free, and beautiful QR code generator built with Next.js, React, and fluid animations.

## ✨ Features
- **Highly Customizable**: Change QR dots style, corner shapes, and corner dots.
- **Custom Logo Upload**: Easily upload your own SVG or PNG logo to be placed perfectly in the center.
- **Text Labels**: Add custom text above and below your QR code, complete with text color customization.
- **Multiple Data Formats**: Supports Website URLs, Plain Text, vCard Contacts, and WiFi network configurations.
- **High-Resolution Export**: Download your final QR code composition in high-res PNG format.
- **Beautiful UI**: Modern glassmorphism design with a dynamic island and fluid, buttery-smooth animations.

## 🛠️ Tech Stack
- **Framework**: Next.js (App Router)
- **Styling**: Vanilla CSS Modules (Glassmorphism & Sky Blue aesthetic)
- **Animations**: Framer Motion
- **QR Engine**: qr-code-styling
- **Capture**: html-to-image

## 🚀 Getting Started

1. **Clone the repository:**
   ```bash
   git clone <repository-url>
   ```
2. **Install dependencies:**
   ```bash
   npm install
   ```
3. **Run the development server:**
   ```bash
   npm run dev
   ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the app in action.
